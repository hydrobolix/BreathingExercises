# Breathing Exercises A–Z

A simple, beautiful single-page website showcasing two complementary A–Z breathing exercise systems.

**Live site:** https://hydrobolix.github.io/breathing-exercises

---

## What's Inside

| Tab | Contents |
|-----|----------|
| **System 1 — Techniques** | 26 named breathing methods (Alternate Nostril → Zone Breathing) |
| **System 2 — Intentions** | 26 goal-oriented practices (Anxiety Relief → Zero Stress) |
| **QCC** | Questions, Concerns & Creative Suggestions — open an issue or email us |

Each tab includes a **Download CSV** button so you can save either list for printing, journaling, or importing into your own apps.

---

## Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks, no build step
- GitHub Pages compatible (deploy straight from the main branch)

---

## Local Development

Just open `index.html` in any browser — no server needed.
