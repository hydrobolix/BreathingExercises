## How This Project Was Built

1. User provided two handwritten lists (photos)
2. Transcribed into structured tables
3. Combined into one cohesive website
4. Generated downloadable CSVs
5. Built with pure HTML/CSS/JS (no frameworks) for maximum simplicity and GitHub Pages compatibility
